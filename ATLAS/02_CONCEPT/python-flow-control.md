- info Permite que o python execute diferentes blocos de código com base em condições especificas 

- ##### [[concept-python-condicionais]]
	- `if`, `elif`, `else`
```python
idade = 18
if idade >= 18:
	print("adulto")
```

- ##### [[loops]]
	- `while` , `for.in` 
```python
i = 0
while i < 5:
	print(i)
	i += 1
```

-  [[concept-python-controle-de-repeticao]]
	- `range` , `break` , `continue`
